Movie Library Project

Steps to Set Up the Project

1.Open the project in VS Code:
Open the project folder (movie-library) in VS Code by navigating to the folder and opening it with VS Code.

2.Open Command Prompt:
Open the Command Prompt (or Terminal on macOS/Linux).

3.Navigate to the Project Folder:
    In the Command Prompt, change the directory to the location of your project. You can use the cd command like this:

cd path\to\movie-library

4.Start the PHP Server:

    After navigating to the project folder, run the following command to start the PHP built-in server:

php -S localhost:8000


    This will start the server and allow you to view the project in your browser by navigating to http://localhost:8000.


** Features of the Movie Library Project

`Search Bar Functionality:

The search bar is fully functional, allowing users to search for movies. The search feature filters and displays relevant results based on user input.

`Mail Sender:

The mail sender feature works as expected, allowing users to send emails from within the application.
